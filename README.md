## 🧩 대표 프로젝트

### 🔗 NewLetsAssemble  
https://github.com/sinhyeongi/NewLetsAssemble_Back  

채팅 읽음 처리에서 발생하는 DB write 집중 문제를 완화하기 위해  
Redis + Batch 구조로 재설계한 백엔드 프로젝트

- 읽음 요청 → Redis → Batch → DB 구조 적용  
- 요청 처리와 DB 반영 시점 분리  
- DB write가 즉시 발생하지 않도록 구조 개선  

---

## 💡 개발 방향

- 요청 처리 흐름을 분석하고 구조적으로 개선  
- DB write 집중을 완화하는 구조 설계  
- Redis 기반 지연 처리 방식 적용
